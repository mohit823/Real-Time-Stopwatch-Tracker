# ⏱️ Real-Time Stopwatch Tracker

A simple and interactive **Stopwatch Web App** built using HTML, CSS, and JavaScript.
This project allows users to track time with features like start, stop, reset, lap recording, and clear functionality.

---

## 🌐 Live Demo

👉 [Click here to view live](https://real-time-stopwatch-tracker.vercel.app/)

---

## 🚀 Features

* ▶️ Start timer
* ⏸️ Stop timer
* 🔄 Reset timer
* 🏁 Record lap time
* 🧹 Clear lap records
* ⏱️ Real-time time tracking
* 🎨 Clean and responsive UI

---

## 🖥️ Preview

* Digital stopwatch display
* Control buttons: Start, Stop, Reset, Lap, Clear
* Lap and stopped time shown below

---

## ⚙️ How It Works

* Timer runs using JavaScript timing functions (`setInterval`)

### Controls:

* **Start** → Begins timer
* **Stop** → Pauses timer
* **Reset** → Sets time to 0
* **Lap** → Records current time
* **Clear** → Removes lap history

## 📦 How to Run Locally

1. Clone the repository:


3. Run the project:
   👉 Open `index.html` (or `stopWatch.html`) in your browser
